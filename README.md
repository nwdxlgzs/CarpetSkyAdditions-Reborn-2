# 修改

## 骆驼和马的新获取途径

移除商人骆驼和骆驼转换机制（`traderCamels`），本机制本来是为了合理刷新骆驼这种生物。在1.21.11群骑纷争中出现了骆驼尸壳，这为骆驼的生成提供了一些可能更为原版感觉的可能。

所以，我们调整了`僵尸马`和`骆驼尸壳`，使其在`虚弱状态`下分别吃`金胡萝卜`和`仙人掌`可以`4000GT`(`200s`)之后变成`马`/`骆驼`
。暂时没有对骷髅马调整的想法，因为僵尸起码还是有肉体的，骷髅马转换为马很奇怪。

我们希望玩家可以游玩高版本内容解锁生物而不是单独的游商变种（另一个原因是版本移植时趁着骆驼尸壳这个机会直接简化移植工作不再修游商和羊驼之间的一堆调整了）。

## 游商的新交易机制调整

由于现版本数据驱动交易，游商附加的新交易移植有所调整，但是为了保证原有设置还有效果（`lavaFromWanderingTrader`/
`tallFlowersFromWanderingTrader`)，现在采用由增加的交易选项线性调整交易概率，将随机替换原有的交易项目。并且这两个设置现在默认为开。

## 数据包空岛调整

新增云杉空岛、大云杉空岛、白桦树空岛、丛林树空岛、大丛林树空岛、深色橡木空岛、杜鹃树空岛、红树空岛、樱花树空岛、苍白橡树空岛，即补全了主世界全部树种的空岛，因为开地狱门就会解锁菌岩，所以没有考虑提供初始地狱的菌柄空岛类型。方便快速切换不同的起始树种进行空岛游玩。

## 甘蔗可催熟

Y=0以下的甘蔗可以无线高度催熟（像竹子一样点一下催熟一点），Y=0以上的甘蔗每次催熟直接长到3节。

## 雪傀儡特殊形态

当启用`existElsaSnowGolem`规则时，名叫`Elsa`或者`艾莎`的雪傀儡可以无视融化环境并且可以冻结脚下的水（根据水源等级冻结成冰、浮冰、蓝冰）。当启用
`existOlafSnowGolem`规则时，名叫`Olaf`或者`雪宝`的雪傀儡可以无视融化环境。

## 服务端伪和平

启动`pseudoPeace`规则后，强制刷怪上限为满上限实现伪和平，该设置旨在为刷怪塔构建阶段提供无需假玩家即可安心构建刷怪塔。

## 禁用唤魔者攻击手段

启动`noAttackEvoker`规则后，唤魔者将无法任何使用攻击手段，但是仍然可以对羊染色。

## 禁用末影人瞬移

启动`banEnderManTeleport`规则后，末影人将无法使用瞬移。
